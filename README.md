# Stone Paper Scissors (Python)

A beginner-friendly Python command-line game where the user plays against the computer.

## Features
- User vs Computer
- Random computer choice
- Input validation
- Replay option

## Technologies Used
- Python
- random module

## How to Run
1. Install Python
2. Run `python game.py`

## Concepts Used
- if-else
- loops
- functions
- boolean logic
